
		  Manipulação de valores
____________________________________________________________

- Para manipular os valores dos inputs vamos utilizar o hook
useState

- Ou seja, podemos armazenar na variável e utilizar o set 
para alterar o valor

- Vamos criar uma função para alterar o valor no evento 
onChange

- Deixando nosso código fácil de trabalhar nas próximas 
etapas: como envio dos dados para BD e validação
